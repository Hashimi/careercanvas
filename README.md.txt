# ProCV Builder - Professional Resume Maker

A modern, bilingual (English/Persian) CV builder with ATS optimization, 10+ templates, and professional features.

## 🚀 Features

### Core Features
- ✅ 10 Professional Templates
- ✅ Bilingual Support (EN/FA) with RTL
- ✅ Profile Picture Upload
- ✅ Save/Load CV Data (JSON)
- ✅ PDF Export (Print)
- ✅ Word Export (.doc)
- ✅ Color Customization
- ✅ Font Selector (4 fonts)

### Professional Features
- ✅ **ATS Score Checker** - Real-time optimization score
- ✅ **Skills with Visual Bars** - Show proficiency levels
- ✅ **AI Writing Assistant** - Auto-generate summaries
- ✅ **Cover Letter Generator** - One-click creation
- ✅ **QR Code Export** - Link to online profiles
- ✅ **Multiple CV Versions** - Save different versions
- ✅ **Auto-Save** - Never lose your work
- ✅ **Certifications Section** - Add professional certs
- ✅ **Social Links** - LinkedIn, GitHub, Portfolio
